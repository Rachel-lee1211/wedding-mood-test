# 16가지 유형 도달 조합 분석

## 핵심 구조 이해

**유형을 결정하는 핵심 질문 3개:**
- **Q1** (꿈꾸는 모습) → 무드 그룹 결정
- **Q9** (듣고 싶은 말) → 무드 재확인
- **Q10** (단 한 번만) → 소재/실루엣 확인

**같은 그룹 내 세부 유형 결정 질문:**
- **Q5** (네크라인) + **Q6** (소재) + **Q7** (홀)
